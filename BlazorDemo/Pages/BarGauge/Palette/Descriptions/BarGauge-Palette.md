The DevExpress Blazor [BarGauge](https://docs.devexpress.com/Blazor/DevExpress.Blazor.DxBarGauge) component allows you to create a custom palette to colorize bars. To apply a color scheme, assign it to the [Palette](https://docs.devexpress.com/Blazor/DevExpress.Blazor.DxBarGauge.Palette) property. If the number of bars exceeds the number of palette colors, you can use the [PaletteExtensionMode](https://docs.devexpress.com/Blazor/DevExpress.Blazor.DxBarGauge.PaletteExtensionMode) property to extend the palette.
 
Use drop-down menus to select a palette and change its extension mode.
 
The [BarGauge](https://docs.devexpress.com/Blazor/DevExpress.Blazor.DxBarGauge) component also allows you to export its data in PNG, PDF, JPEG, and SVG [file formats](https://docs.devexpress.com/Blazor/DevExpress.Blazor.DataExportFormat). To enable export functionality, add a [DxExportSettings](https://docs.devexpress.com/Blazor/DevExpress.Blazor.DxExportSettings) object to gauge markup and set the [Enabled](https://docs.devexpress.com/Blazor/DevExpress.Blazor.DxExportSettings.Enabled) property to `true`. You can also configure settings of the exported file ([FileName](https://docs.devexpress.com/Blazor/DevExpress.Blazor.DxExportSettings.FileName), [Margin](https://docs.devexpress.com/Blazor/DevExpress.Blazor.DxExportSettings.Margin), and [BackgroundColor](https://docs.devexpress.com/Blazor/DevExpress.Blazor.DxExportSettings.BackgroundColor) properties) and enable/disable [printing](https://docs.devexpress.com/Blazor/DevExpress.Blazor.DxExportSettings.PrintingEnabled).
 
Click the **Enable Export** check box to display the export hamburger menu.
