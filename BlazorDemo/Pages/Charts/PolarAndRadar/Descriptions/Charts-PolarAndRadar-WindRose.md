The DevExpress Blazor [Polar Chart](https://docs.devexpress.com/Blazor/DevExpress.Blazor.DxPolarChart-1) in this demo uses a [stacked bar series](https://docs.devexpress.com/Blazor/DevExpress.Blazor.DxPolarChartStackedBarSeries-3) to display a wind rose. The chart component configures the argument axis as follows:

* Displays axis labels next to major ticks and grid lines (the [DiscreteDivisionMode](https://docs.devexpress.com/Blazor/DevExpress.Blazor.DxChartAxisBase-1.DiscreteDivisionMode) is set to `CrossLabels`).
* Aligns the first chart point to the value axis (the [FirstPointOnStartAngle](https://docs.devexpress.com/Blazor/DevExpress.Blazor.DxPolarChartArgumentAxis.FirstPointOnStartAngle) property is enabled).
