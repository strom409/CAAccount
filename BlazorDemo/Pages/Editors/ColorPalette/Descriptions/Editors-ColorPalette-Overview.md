The DevExpress Blazor [Color Palette](https://docs.devexpress.com/Blazor/DevExpress.Blazor.DxColorPalette) UI component allows users to select color values within your DevExpress-powered Blazor app (the palette can display multiple color [groups](https://docs.devexpress.com/Blazor/DevExpress.Blazor.DxColorPalette#color-groups) - [predefined sets](https://docs.devexpress.com/Blazor/DevExpress.Blazor.DxColorPalette#palette-presets) or [custom colors](https://docs.devexpress.com/Blazor/DevExpress.Blazor.DxColorPalette#custom-colors)).

Our Color Palette UI component supports different size mode options. To specify component size in code, use the [SizeMode](https://docs.devexpress.com/Blazor/DevExpress.Blazor.Base.DxResizableEditorBase-2.SizeMode) property. To apply different size modes to this demo, use the dropdown list in the demo card header.

The Color Palette also supports keyboard navigation ([list of supported keyboard shortcuts](https://docs.devexpress.com/Blazor/DevExpress.Blazor.DxColorPalette#keyboard-navigation)), allowing users to navigate within the palette and select individual colors.

This demo uses our default Color Palette configuration (includes three color groups: **Universal**, **Universal Gradient**, and **Standard**).
