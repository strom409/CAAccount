Our Blazor [Date Range Picker](https://docs.devexpress.com/Blazor/DevExpress.Blazor.DxDateRangePicker-1) component allows you to select a date range using a single calendar dropdown.

The DevExpress Blazor Date Range Picker component supports multiple size modes. To specify component size in code, use the [SizeMode](https://docs.devexpress.com/Blazor/DevExpress.Blazor.Base.DxResizableEditorBase-2.SizeMode) property. To apply different size modes within this demo, use the drop-down list in the demo card's header.

The Date Range Picker supports keyboard navigation ([list of supported keyboard shortcuts](https://docs.devexpress.com/Blazor/DevExpress.Blazor.DxDateRangePicker-1#keyboard-navigation)), allowing users to navigate to the editor's input element and within the drop-down calendar.
