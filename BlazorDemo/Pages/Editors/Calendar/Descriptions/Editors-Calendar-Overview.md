The DevExpress Blazor [Calendar](https://docs.devexpress.com/Blazor/DevExpress.Blazor.DxCalendar-1) component allows users to select dates and navigate through months, years, decades, and centuries.

The Calendar supports keyboard navigation ([list of supported keyboard shortcuts](https://docs.devexpress.com/Blazor/DevExpress.Blazor.DxCalendar-1#keyboard-navigation)), allowing users to navigate through root UI elements and within the month/year/decade/century view.
