Our [Rich Text Editor](https://www.devexpress.com/blazor/rich-text-editor/) for Blazor allows you to edit and format documents in code.

Review the code in this demo modules and learn how to do the following:

* Populate an open document with [content](https://docs.devexpress.com/Blazor/DevExpress.Blazor.RichEdit.DxRichEdit#document-api) (text, image, and links)
* Create and customize [tables](https://docs.devexpress.com/Blazor/404904/components/rich-edit/examples/tables)
* [Format text](https://docs.devexpress.com/Blazor/403347/components/rich-edit/examples/text-operations#format-text)
* Change [paragraph settings](https://docs.devexpress.com/Blazor/DevExpress.Blazor.RichEdit.ParagraphProperties)
