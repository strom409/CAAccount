The DevExpress Blazor [Drop-Down Button](https://docs.devexpress.com/Blazor/DevExpress.Blazor.DxDropDownButton) component can display multi-level drop-down lists. Populate a command's [Items](https://docs.devexpress.com/Blazor/DevExpress.Blazor.DxDropDownButtonBase.Items) property with [DxDropDownButtonItem](https://docs.devexpress.com/Blazor/DevExpress.Blazor.DxDropDownButtonItem) objects to create a nested menu. 

Use the [BeginGroup](https://docs.devexpress.com/Blazor/DevExpress.Blazor.DxDropDownButtonItem.BeginGroup) property to separate items into groups (at any nesting level).

Our Blazor Drop-Down Button component can also display icons within the main button and for drop-down list items. Use [IconCssClass](https://docs.devexpress.com/Blazor/DevExpress.Blazor.DxButtonBase.IconCssClass) and [IconPosition](https://docs.devexpress.com/Blazor/DevExpress.Blazor.DxButtonBase.IconPosition) properties to configure icon settings.
