When progress cannot be estimated or it is not necessary to indicate progress numerically, set the [Indeterminate](https://docs.devexpress.com/Blazor/DevExpress.Blazor.DxProgressBar.Indeterminate) property to `true` to display a moving bar.

The default progress bar label displays information about progress status. You can use the following properties to customize the label as needs dictate.

* The [Label](https://docs.devexpress.com/Blazor/DevExpress.Blazor.DxProgressBar.Label) property: specifies the progress bar label.
* The [ShowLabel](https://docs.devexpress.com/Blazor/DevExpress.Blazor.DxProgressBar.ShowLabel) property: allows you to hide the label from the progress bar.
* The [ShowIcon](https://docs.devexpress.com/Blazor/DevExpress.Blazor.DxProgressBar.ShowIcon) property: allows you to display an icon in the progress bar.
