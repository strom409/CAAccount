DevExpress Blazor [Map](https://docs.devexpress.com/Blazor/DevExpress.Blazor.DxMap) component allows you to display and configure [map markers](https://docs.devexpress.com/Blazor/DevExpress.Blazor.DxMapMarker). To create a marker, complete the following steps:

1. Add a [DxMapMarker](https://docs.devexpress.com/Blazor/DevExpress.Blazor.DxMapMarker) object to the [marker collection](https://docs.devexpress.com/Blazor/DevExpress.Blazor.DxMapMarkers).
2. Use a [DxMarkerLocation](https://docs.devexpress.com/Blazor/DevExpress.Blazor.DxMapMarkerLocation) object to specify marker position on the map.
3. *Optional*. Customize marker [appearance](https://docs.devexpress.com/Blazor/DevExpress.Blazor.DxMapMarker.IconUrl) and configure [tooltip](https://docs.devexpress.com/Blazor/DevExpress.Blazor.DxMapMarkerTooltip) settings.

This demo allows you to create, modify, and remove map markers. Click the map or a marker to display the *Marker Editor* dialog.
