Our new Blazor [Pivot Table](https://docs.devexpress.com/Blazor/DevExpress.Blazor.PivotTable.DxPivotTable) UI component (Community Technology Preview - official ship date June 2025) summarizes data from the underlying data source and presents results using a cross-tabulated UI metaphor. It ships with the following integrated features/capabilities:

* High-performance data engine
* [Data sorting support](https://docs.devexpress.com/Blazor/DevExpress.Blazor.PivotTable.DxPivotTable#sort-data)
* DateTime, numeric, alphabetic, and custom [group intervals](https://docs.devexpress.com/Blazor/DevExpress.Blazor.PivotTable.DxPivotTable#group-data)
* Configurable [totals and grand totals](https://docs.devexpress.com/Blazor/DevExpress.Blazor.PivotTable.DxPivotTable#totals-grand-totals)
* Built-in scroll bar and sticky Header/Row Value areas
* [Virtual scrolling support](https://docs.devexpress.com/Blazor/DevExpress.Blazor.PivotTable.DxPivotTable#virtual-scrolling)
* [Value formatting](https://docs.devexpress.com/Blazor/DevExpress.Blazor.PivotTable.DxPivotTable#data-format) and [integrated templates](https://docs.devexpress.com/Blazor/DevExpress.Blazor.PivotTable.DxPivotTable#templates)
 
In this demo, the Pivot Table allows you to browse/analyze prices and discounts across different car models (by quarter/year). It also calculates and displays sub-totals and grand totals for relevant data points.
